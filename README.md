
# ATM Simulator using OOPs in Python

This project is a console-based ATM Simulator built using Object-Oriented Programming concepts in Python.

## Features
- Create PIN
- Change PIN
- Check balance
- Withdraw money
- PIN authentication

## OOP Concepts Used
- Class & Object
- Encapsulation
- Methods
- State management

## How to Run
```bash
python atm.py
